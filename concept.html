Photography

  • Ideation, Graphic design
  • Tools: Photoshop, Illustrator
  • date: Summer 2018
  • Personal Project

The main intention of this project was to think out of the box and create visually interactive design.

Ideation

It all began with trying out new ideas and a lot of time put in masking different photos together, but ultimately it was fun.

These were all indivisual images among which some of them were captured and rest were royalty free images, I masked together in order to form an isometric view.

It was in downtown Vancouver where I clicked this image and won a photography competition in Fraser International College.

Reflections

This was a personal project which helped me to create image compositions creatively. Moreover, the use of Photoshop taught me a lot of things that will allow me to use these skills in further projects.
